# Приложение для построения генеалогического дерева

## Описание проекта
Приложение для создания и управления семейным (генеалогическим) деревом, реализованное на C# с использованием Windows Forms. Позволяет добавлять/удалять членов семьи, устанавливать родственные связи и визуализировать дерево в графическом интерфейсе.

<img width="1630" height="407" alt="image" src="https://github.com/user-attachments/assets/71ee7972-b6d6-4c6c-a58d-c14e3271a85e" />


## Функциональность
- Добавление, удаление и редактирование членов семьи
- Управление родственными связями (родитель–ребёнок, супруги и др.)
- Визуализация семейного дерева
- Графический интерфейс

## Технологии
- **Язык программирования**: C#
- **GUI**: Windows Forms
- **Визуализация связей**: Graphviz
- **Принципы**: ООП, строгая типизация
